# 💰 Daily Expense Tracker

**100% free, open-source daily expense tracker with charts, categories, recurring transactions, budget goals, and CSV export.**

No signup. No tracking. No ads. All data stays on your device.

[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

[![Free Forever](https://img.shields.io/badge/💰-Free%20Forever-orange.svg)](#)

## 🚀 Features

### Free Features
- ✅ **Unlimited transactions** — track income and expenses
- ✅ **9 categories** — Food, Transport, Entertainment, Shopping, Bills, Health, Education, Savings, Other
- ✅ **Interactive charts** — doughnut chart (spending breakdown) + bar chart (7-day trend)
- ✅ **Monthly summary** — category breakdown with progress bars
- ✅ **Filter & search** — by date range, category, or keyword
- ✅ **CSV export** — download your data as a spreadsheet
- ✅ **JSON backup/restore** — full data backup and import
- ✅ **3 recurring rules** — subscriptions, rent, salary — set once, tracked forever
- ✅ **1 budget goal** — monthly spending limits with progress tracking
- ✅ **Keyboard shortcuts** — Enter to submit, Tab to navigate
- ✅ **Responsive design** — works on desktop, tablet, and mobile
- ✅ **Dark theme** — easy on the eyes

### Pro Features ($4.99/mo)
- 🔒 Unlimited recurring rules
- 🔒 Unlimited budget goals with alerts
- 🔒 Cloud sync (Supabase)
- 🔒 AI spending insights
- 🔒 Multi-device sync
- 🔒 Priority support

## 📦 Quick Start

### Option 1: Open Directly
1. Download `daily_expense_tracker.html`
2. Open it in any modern browser
3. Start tracking!

### Option 2: Clone & Serve
```bash
git clone https://github.com/adbhutrd/expense-tracker.git
cd expense-tracker
python3 -m http.server 8000
# Open http://localhost:8000/daily_expense_tracker.html
```

## 🛡️ Privacy & Security

- **100% client-side** — no server, no database, no cloud
- **localStorage only** — data never leaves your browser
- **No tracking** — zero analytics, zero cookies
- **No ads** — clean, distraction-free experience
- **Open source** — full code transparency

## 🏗️ Tech Stack

- **HTML5** — semantic markup
- **CSS3** — custom properties, grid, flexbox, animations
- **JavaScript (ES6+)** — vanilla JS, no frameworks
- **Chart.js** — interactive charts and visualizations
